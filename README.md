# Outlook Spam Detector (GitHub Hosted)

✔ No paid software  
✔ No dataset downloads  
✔ Trains in GitHub  
✔ Runs locally on Outlook  

## How it works
- Train model using GitHub Codespaces:
python src/train.py
- This generates:
- `models/spam_model.pkl`
- `models/vectorizer.pkl`

- Run Outlook scanner:
python src/outlook_scanner.py
